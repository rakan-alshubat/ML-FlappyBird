Enter into the command line (in the same directory as the file) 'python main.py' followed by either the word 'play' or 'train' depending on which one you would like it to do, followed by an integer of how many times it should do it, and finally, the name of the output file the data should go into/read from.

Main < play | train > < int game count > < string file name output or load >
